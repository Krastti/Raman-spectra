> **Задача:** Разработка решения на основе машинного обучения для классификации рамановских спектров биологических тканей с целью определения экспериментальной группы: **Контроль**, **Эндогенная экспрессия HSP70**, **Экзогенная экспрессия HSP70**.

---

## 📖 О проекте

Рамановская спектроскопия — мощный инструмент для неразрушающего анализа биологических образцов. Однако данные часто зашумлены, содержат флуоресцентный фон и обладают высокой размерностью. 

Этот репозиторий содержит полное решение задачи классификации спектров, включая предобработку, извлечение признаков, обучение моделей и интерпретацию результатов. Решение оптимизировано для работы с большими датасетами, устойчиво к шумам и предоставляет биологически интерпретируемые результаты.

### 🎯 Цели
1. **Автоматизация:** Создать воспроизводимый пайплайн обработки спектров.
2. **Точность:** Максимизировать метрики классификации (Accuracy, F1, ROC-AUC).
3. **Интерпретируемость:** Выявить спектральные области (биомаркеры), значимые для различения групп.

## 🧪 Методология (Pipeline)

Наш подход состоит из 4 ключевых этапов, направленных на борьбу с высокой размерностью и шумом.

### 1. Предобработка (Preprocessing)
Для устранения артефактов измерения применяются:
*   **Удаление шума:** Фильтр Савицкого-Голея (Savitzky-Golay) с подбором окна.
*   **Коррекция базовой линии:** Алгоритм Asymmetric Least Squares (ALS) для удаления флуоресцентного фона.
*   **Нормировка:** Vector Normalization или Area Under Curve (AUC) для приведения интенсивностей к общему масштабу.

### 2. Извлечение признаков (Feature Engineering)
*   **Спектральные пики:** Автоматическое выделение пиков и их интегральных интенсивностей.
*   **Производные:** Использование 1-й и 2-й производных для улучшения разделимости перекрывающихся пиков.
*   **Снижение размерности:** 
    *   PCA (Principal Component Analysis) для линейного сжатия.
    *   Autoencoders (PCAe) для нелинейного выделения латентных признаков.

### 3. Моделирование (Modeling)
Сравнение ансамблевых и нейросетевых подходов:
*   **Baseline:** Logistic Regression, SVM (RBF kernel).
*   **Advanced:** Random Forest, XGBoost, LightGBM.
*   **Deep Learning:** 1D-CNN или MLP на латентных признаках автоэнкодера.

### 4. Валидация и Оценка
*   **Кросс-валидация:** Stratified K-Fold (k=5/10) для сохранения баланса классов.
*   **Метрики:** Accuracy, Weighted F1-Score, ROC-AUC (One-vs-Rest).
*   **Балансировка:** SMOTE или класс-весы при дисбалансе групп.
